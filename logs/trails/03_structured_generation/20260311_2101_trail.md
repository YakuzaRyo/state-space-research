## 研究流程执行

### Step 1: Web Research (8-10分钟)

#### XGrammar论文核心发现

**论文**: [XGrammar: Flexible and Efficient Structured Generation Engine for Large Language Models](https://arxiv.org/abs/2411.15100) (陈天奇团队, CMU, 2024年11月)

**核心创新**:
1. **Token分类策略**: 将token分为上下文无关(>99%)和上下文相关(<1%)
2. **自适应Token Mask缓存**: 通过智能存储格式将内存从160MB降至0.46MB
3. **字节级PDA**: 直接在字节级别处理，支持不规则token边界
4. **持久执行栈**: 树形数据结构支持O(1)回滚和状态分支

**性能数据**:
- Token Mask生成时间: < 40微秒
- 相比现有方案: 100倍加速
- H100 GPU端到端: 80倍吞吐提升

#### Outlines库分析

**核心机制**:
- 使用FSM (有限状态机) 进行约束解码
- 在token级别通过logit masking限制输出
- 支持JSON Schema、正则表达式、多选约束

**限制**:
- 仅支持正则语言（无法处理嵌套结构）
- 对于复杂CFG需要递归下降

#### LM Format Enforcer实现

**双树架构**:
```
字符级解析器 ←→ Tokenizer前缀树(Trie)
         ↓
   有效Token = 交集
```

**特点**:
- Python实现，可读性强
- 动态计算token masks
- 相比Rust实现有性能差距

#### llguidance (Microsoft)

**技术特点**:
- Rust实现，~50μs/token
- 使用Earley Parser而非PDA
- 被OpenAI Structured Outputs采用

#### 核心模块验证

**1. TokenBitmask - 高效掩码存储**
```rust
pub const VOCAB_SIZE: usize = 50000;
pub const BITMASK_WORDS: usize = (VOCAB_SIZE + 64 - 1) / 64; // = 782

pub struct TokenBitmask {
    words: [u64; BITMASK_WORDS], // 约6.2KB存储
}
```

验证点:
- `allow_token()` / `disallow_token()` - 单token操作
- `is_allowed()` - O(1)查询
- `intersect()` - 掩码交集(位运算AND)
- `apply_to_logits()` - 应用到logits数组

**2. TokenClassifier - Token分类器**
```rust
pub enum TokenClass {
    ContextIndependent,  // >99%
    ContextDependent,    // <1%
}
```

验证点:
- 可标记特定token为context-dependent
- 计算context-dependent比例
- 测试显示2/1000 = 0.2%符合预期

**3. PdaStack - 下推自动机栈**
```rust
pub struct PdaStack {
    stack: Vec<StackElement>,
}

pub struct StackElement {
    pub return_state: u32,
    pub return_rule: u32,
    pub return_position: u16,
}
```

验证点:
- push/pop操作
- peek查看栈顶
- depth计算栈深度
- is_empty检查

**4. ContextIndependentCache - 上下文无关缓存**
```rust
pub struct ContextIndependentCache {
    cache: HashMap<PdaPosition, TokenBitmask>,
}
```

验证点:
- 插入/查询缓存
- PdaPosition作为key (state_id, rule_id, position)

**5. ConstraintEngine - 约束引擎**
```rust
pub struct ConstraintEngine {
    ci_cache: ContextIndependentCache,
    classifier: TokenClassifier,
    stack: PdaStack,
    current_pos: PdaPosition,
    rules: Vec<GrammarRule>,
}
```

核心方法:
- `compute_mask()` - 计算当前状态的token掩码
- `advance(token)` - 根据生成token更新状态
