## 1. Web Research 阶段 (8-10分钟)

### 关键发现

#### 1.1 LLM-A* 论文 (EMNLP 2024)
- **论文**: LLM-A*: Large Language Model Enhanced Incremental Heuristic Search on Path Planning
- **作者**: Silin Meng et al. (UCLA)
- **arXiv**: 2407.02511

**核心贡献**:
- 结合A*算法的精确局部搜索与LLM的全局上下文理解
- 修改启发式函数: `h_LLM-A*(n) = h_A*(n) + c_LLM(n)`
- `c_LLM(n)` 包含预测的转移成本、障碍物规避、环境复杂度

**理论要点**:
> "While this enhanced heuristic expedites the search process... it introduces a deviation from admissibility. By incorporating the additional cost c_LLM(n), the heuristic may overestimate the true cost to reach the goal."

**结论**: LLM-A* 明确承认**偏离可采纳性**，以牺牲最优性换取计算效率。

#### 1.2 学习可采纳启发式 (Learning Admissible Heuristics)
- **论文**: Learning Admissible Heuristics with Neural Networks (Li, 2022, University of Alberta)
- **核心问题**: 神经网络启发式通常是**不可采纳的**（高估成本）

**解决方案**:
1. 学习分类器
2. 调整分类器分位数
3. 取神经网络集合的最小值

#### 1.3 近似可采纳启发式 (Approximately Admissible)
- **论文**: Obtaining Approximately Admissible Heuristic Functions through Deep RL and A* Search (Agostinelli et al., 2021)
- **成果**: 在15-puzzle和24-puzzle上实现**99.99%经验可采纳性**
- 找到100%测试用例的最优解

#### 1.4 交叉熵可采纳性 (Cross-Entropy Admissibility)
- **论文**: Learning Admissible Heuristics for A*: Theory and Practice (2025)
- **创新**: CEA损失函数在训练期间**强制执行可采纳性**
- 提供学习启发式的理论样本复杂度界限

## 2. 假设提出 (3-5分钟)

### 核心假设

**H1: LLM作为启发式函数本质上是一个学习得到的、概率性的启发式估计器**

- LLM通过预训练学习了大量关于世界结构和问题求解的模式
- 这种学习是隐式的、基于统计的，而非显式编程
- 因此LLM启发式类似于神经网络学习的启发式

**H2: LLM启发式通常不满足可采纳性，但满足"近似可采纳性"**

- 由于LLM可能基于模式识别高估或低估成本
- 但在大规模预训练下，高估的概率可以被控制在一定范围内
- 可以定义"ε-可采纳性": P(h(n) ≤ h*(n)) ≥ 1-ε

**H3: LLM启发式的优势在于"语义理解"而非"数值精确性"**

- LLM擅长理解问题的语义结构（如"这个方向看起来更有希望"）
- 传统启发式擅长精确的数值估计（如欧几里得距离）
- 混合方法（如LLM-A*）结合两者优势

**H4: LLM启发式的理论基础可以建立在"经验可采纳性"和"概率最优性"之上**

- 传统A*要求严格的数学可采纳性
- LLM启发式可以提供概率保证：以高概率找到接近最优的解
- 这类似于近似算法和随机算法的理论框架

## 4. 输出结果 (5-8分钟)

### 4.1 理论框架总结

LLM作为启发式函数的理论基础可以建立在以下支柱上：

1. **统计学习理论**: LLM是通过大规模预训练学习得到的启发式估计器
2. **近似可采纳性**: 定义ε-可采纳性和概率最优性
3. **混合架构**: 结合传统可采纳启发式与LLM语义理解
4. **经验验证**: 通过大规模实验验证启发式质量

### 4.2 关键公式

**ε-可采纳性定义**:
```
P(h_LLM(n) ≤ h*(n)) ≥ 1-ε, ∀n ∈ S
```

**混合启发式** (LLM-A*风格):
```
h_hybrid(n) = α · h_admissible(n) + β · h_LLM(n)
```
其中α + β = 1, h_admissible保证基础可采纳性

**期望次优界**:
```
E[cost(path) - cost(optimal)] ≤ δ
```
