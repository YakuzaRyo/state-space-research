#### 查询6: 神经启发式理论性质
**关键词**: "neural heuristic function admissibility consistency theoretical properties state space search"

**关键发现**:
- **ε-Consistency (ICLR 2025)**: 神经启发式的松弛一致性
  - 定义: h(x) ≤ ε·V(x,y) + h(y) for any ε > 1
  - 边界: ε = max_{x≠y} 1 + 2ε_NO/V(x,y)

- **Approximately Admissible Heuristics (Agostinelli et al., 2021)**:
  - 基线: 神经网络启发式高估30-71%
  - 转换后: 99.99%经验可采纳性
  - 结果: 100%最优解率 (15-puzzle, 24-puzzle)

## Step 2: 提出假设 (3-5分钟)

基于Web Research，提出以下理论假设：

### H1: ε-Admissibility假设
**陈述**: LLM启发式满足ε-可采纳性

**形式化**:
```
h_LLM(s) ≤ h*(s) + ε for all states s
```

**论证**:
- 通过保守提示("estimate minimum steps")可以降低高估概率
- 校准技术可以调整LLM输出分布
- Agostinelli et al. (2021) 证明了近似可采纳性的可行性

### H2: 语义一致性假设
**陈述**: LLM启发式在语义相似状态间保持一致

**形式化**:
```
|h_LLM(s1) - h_LLM(s2)| ≤ δ · d_semantic(s1, s2)
```

**论证**:
- LLM基于语义理解进行估计
- 相似状态应有相似难度评估
- 这是LLM相对于传统启发式的优势

### H3: 信息-计算权衡假设
**陈述**: LLM启发式提供更好的信息性，代价是计算开销

**形式化**:
```
(T_analytical × N_analytical) > (T_LLM × N_LLM)
```

**论证**:
- 在复杂状态空间，LLM的语义理解可以减少搜索节点数
- 当状态空间复杂度超过阈值时，LLM启发式更优

### H4: 有效分支因子降低假设
**陈述**: LLM先验策略降低有效分支因子

**形式化**:
```
b_eff = Σ_a P_LLM(a) · indicator(a is useful) << b_raw
```

**论证**:
- LLM-MCTS证明LLM可以作为有效的policy prior
- 语义理解帮助识别promising actions
- AlphaProof的progressive sampling验证了这一点
